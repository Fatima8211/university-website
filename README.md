## ✨ Features

- ✅ **Fully Responsive** - Works perfectly on all devices (mobile, tablet, desktop)
- ✅ **Interactive Navigation** - Dropdown menu with smooth animations
- ✅ **Modern UI/UX** - Clean, professional design with consistent branding
- ✅ **6+ Pages** - Home, About, News, Departments, Contact, Registration
- ✅ **Department Pages** - Dedicated pages for each department
- ✅ **Course Showcase** - Display of undergraduate, graduate, and doctoral programs
- ✅ **Campus Gallery** - Beautiful campus images with hover effects
- ✅ **Facilities Section** - Sports, Library, Cafeteria with overlay effects
- ✅ **Student Testimonials** - With star ratings
- ✅ **Online Courses** - Call-to-action section
- ✅ **Contact Form** - With validation
- ✅ **Registration Form** - Complete student registration
- ✅ **Social Media Integration** - Instagram, Facebook, WhatsApp links
